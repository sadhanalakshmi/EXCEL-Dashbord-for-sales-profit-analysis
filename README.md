# EXCEL-Dashbord-for-sales-profit-analysis

OVERVIEW

The Sales and Profit Analysis Dashboard was developed using Microsoft Excel to provide an interactive and visual representation of business performance. The dashboard enables users to analyze sales and profit data across different regions, months, and years through dynamic charts and slicers.

OBJECTIVE

Monitor  overall sales and profit performance and Compare sales and profit across regions.
Analyze monthly,yearly sales and profit trends.
Support data-driven decision-making through interactive visualizations.
